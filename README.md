# Ordena24 API

API REST para el sistema web de pedidos a domicilio del restaurante Ordena24.

## Tecnologías

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Uvicorn

## Funcionalidades iniciales

- Gestión de productos
- Gestión de categorías
- Gestión de pedidos
- Actualización del estado de pedidos
